ADCMT 7351 Instrument Driver (ADC command) Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: ADC Corporation
Supported Language(s): LabVIEW8.0
Supported Model(s): ADCMT 7351A/E
Model(s) Tested: ADCMT 7351A/E
Interface(s): GPIB, USB
Firmware Revision(s) Tested: C01

Certified: No
NI Supported: No
Source Code Available: Yes

Driver Revision: 1.0.0
Original Release Date: 07/10/2008
Current Revision Date: 07/10/2008

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 4.0 or later
LabVIEW 8.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to kcc@adcmt.

This driver does not support SCPI mode.
