ADCMT 6240 Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: ADC Corporation
Supported Language(s): LabVIEW
Supported Model(s): 6240A, 6240B
Model(s) Tested: 6240A, 6240B
Interface(s): GPIB
Firmware Revision(s) Tested: A00 (6240A), A00 (6240B)

Certified: No
NI Supported: No
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 08/27/2008
Current Revision Date: 02/27/2015

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 4.0 or later
LabVIEW 8.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to kcc@adcmt.com.

4. Revision History

The latest version of this instrument drivers can be downloaded at the ADCMT LabVIEW Support.
